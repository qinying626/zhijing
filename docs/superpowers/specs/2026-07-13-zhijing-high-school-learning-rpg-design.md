# 知境 — 高中游戏化学习平台设计

## 概述

知境（ZhiJing）是一个面向高中生的游戏化学习网站，覆盖数学、语文、英语、地理、物理五大学科。以 RPG 冒险为核心玩法，将高中知识体系和近10年高考真题融入游戏机制，让学生在探索与战斗中学习。支持多人在线互动（组队、PK、排行榜）。

## 世界观

玩家扮演"求索者"，进入由知识构建的"知境大陆"。大陆由五大领域组成，每个领域对应一门学科：

- **数理殿** — 数学（逻辑与秩序之域）
- **文心阁** — 语文（文字与意境之域）
- **灵语林** — 英语（跨语言沟通之域）
- **天地图** — 地理（山川与文明之域）
- **万象庭** — 物理（自然法则之域）

大陆中央是"知行广场"，为多人社交与交易枢纽。大陆外围散布"时空裂隙"——高考真题副本入口。

## 核心玩法

### 角色系统

- 创建角色时选择初始学科倾向（不影响后续学习所有学科）
- 等级由经验值驱动，经验值来源于答题、通关、日常任务
- 每个学科独立等级和技能树，鼓励全面发展

### 战斗与答题

- 遇到怪物/关卡 → 触发答题界面
- 答对：造成伤害/通过关卡，获得经验和掉落
- 答错：受到伤害/关卡失败，进入"错题回溯"可重新挑战
- 难度分级：基础（小怪）→ 进阶（精英怪）→ 挑战（Boss）

### 日常与周常

- 每日答题任务（每日签到 + 5道随机题）
- 每周学科挑战（限时副本，排名奖励）

## 学科特色玩法

### 数学 — 数理殿

- **推演系统**：每道数学题拆解为多步推导，每步答对推进"公式链"，答错链条断裂需重新推导
- **几何画板**：几何题提供可交互的图形工具，拖拽验证猜想
- **Boss 战**：综合大题，需连续完成多步骤推导

### 语文 — 文心阁

- **诗词拼图**：古诗词按句打乱，拖拽排列正确顺序
- **文言解码**：文言文逐句翻译，选择正确的现代文释义
- **作文擂台**：限时写作，AI 辅助评分 + 同学互评
- **阅读探索**：现代文阅读理解，按段落逐步解锁问题

### 英语 — 灵语林

- **词汇收集**：背单词像收集卡牌，每个单词是一张卡，集齐主题套组触发奖励
- **语法阵法**：语法题以"排列阵法"形式呈现，拖拽词块组成正确句式
- **听力迷雾**：听音频在迷雾地图中寻路，听懂方向才能前进
- **完形填空桥梁**：完形填空题表现为修复断裂的桥梁，每填一词修复一段

### 地理 — 天地图

- **地图探索**：在交互式地图上定位地理现象（气候带、洋流、城市分布）
- **沙盘推演**：模拟自然现象（板块运动、水循环、城市化进程）
- **区域解锁**：答对某区域相关题目后解锁该地图区域，收集地理信息

### 物理 — 万象庭

- **实验工坊**：模拟物理实验（力的合成、电路搭建、光学折射），参数可调观察结果
- **公式锻造**：用已知公式组合推导新公式，类似合成系统
- **现象还原**：描述物理现象 → 选择正确原理解释

## 高考真题系统（近10年 2016-2025）

### 时空裂隙副本

- 每年高考对应一个"时空裂隙"副本
- 进入后面对该年真题，按学科分区
- 真题来源：全国卷(I/II/III)、新高考卷、各省自主命题卷

### 真题融入方式

- **拆步作答**：大题拆成小步骤，每步答对获得积分和掉落
- **全服排行**：同一道真题的正确率、用时排行
- **错题集**：答错的真题自动收录，支持重做和查看解析

### 筛选与刷题

- 按年份、省份、学科、难度筛选真题
- 支持随机组卷模式（模拟考试体验）

## 多人在线系统

### 社交

- 好友系统、私聊、组队频道
- 学科讨论区（按学科分区）

### 协作与竞争

- **组队副本**：2-4人组队挑战综合题副本，每人负责不同部分
- **学科 PK**：1v1 限时答题对战
- **公会/学习小组**：创建或加入学习小组，共享进度和资源

### 排行榜

- 全服等级榜、单科等级榜
- 真题正确率榜、周挑战排名
- 学校/班级排行榜（可选加入）

## 知识内容体系

### 模块化设计

- 每个知识点为独立数据单元，包含：知识卡片（概念讲解）、技能训练（练习题）、Boss 战（综合题）
- 知识点按学科 → 领域 → 章节 → 知识点四级组织
- 初期搭建框架并提供少量示例内容，后续可扩展

### 可扩展性

- 知识点通过管理后台或内容导入工具持续补充
- 题库支持批量导入（JSON/CSV 格式）
- 题目支持多种题型：选择题、填空题、简答题、拖拽排序、交互操作

### 前端

- **React 18** + TypeScript
- **Phaser.js** — 2D 游戏引擎（地图探索、战斗动画、交互式实验）
- **Zustand** — 状态管理
- **Tailwind CSS** — UI 样式
- **Socket.io-client** — 实时通信

### 后端微服务

| 服务 | 技术栈 | 职责 |
|------|--------|------|
| 用户服务 | Node.js + Express + PostgreSQL | 注册/登录、角色管理、等级经验 |
| 学科服务 | Node.js + Express + MongoDB | 各学科知识点、题库、答题逻辑 |
| 游戏服务 | Node.js + Colyseus + PostgreSQL | RPG 战斗、副本、掉落、技能树 |
| 社交服务 | Node.js + Express + MongoDB | 好友、聊天、组队、PK、排行榜 |
| 内容服务 | Node.js + Express + MongoDB | 知识卡片、真题管理、内容导入 |

### 实时通信

- **Colyseus** — 游戏房间管理（组队副本、PK 对战）
- **Socket.io** — 社交实时消息（聊天、通知）
- **Redis** — 会话缓存、排行榜、实时数据

### 部署

- Docker + Docker Compose（开发环境）
- Kubernetes（生产环境）
- CI/CD: GitHub Actions

## MVP 范围

MVP 阶段聚焦核心循环验证，不含全部功能：

1. 单学科（数学）完整体验：知识卡片 → 练习 → Boss 战
2. 时空裂隙副本（2年真题示例）
3. 基础角色系统（创建、升级、经验值）
4. 好友与排行榜
5. 管理后台（题目导入）

MVP 之外的功能（后续迭代）：
- 其余四学科玩法
- 组队副本、PK 对战
- 公会/学习小组
- 高级交互（几何画板、物理实验、地理沙盘）
- 听力系统
- AI 作文评分
